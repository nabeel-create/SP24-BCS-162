# Demo Video Script (2-4 minutes)

1. **Intro**
   - Open Dashboard.
   - Mention: tasks, categories, repeats, exports, notifications.

2. **Dashboard Highlights**
   - Show Overall Progress.
   - Use Quick Search under the progress card.
   - Show Pinned section.

3. **Add Task**
   - Create a task with title, description, category, priority.
   - Pick start/end dates, set repeat days.
   - Use the Pin chip (Normal/Pinned).

4. **Subtasks + Progress**
   - Add 2 subtasks.
   - Mark one completed and show progress update.

5. **Filters + Sort**
   - Go to Tasks screen.
   - Show Today, Completed, Repeated, Overdue filters.
   - Use Sort by (Pinned, Date, Priority).

6. **Categories**
   - Open Categories screen.
   - Tap a category to open its task list.
   - Show completed/pending counts on each card.

7. **Overdue Flow**
   - Open an overdue task.
   - Tap complete -> confirm dialog (Complete/Delete).

8. **Export + Auto-Archive**
   - Settings -> Export CSV and PDF.
   - Toggle Auto-archive completed (7 days).

9. **Notifications**
   - Show FCM push test from Firebase Console.
   - Mention OEM limits for background local alarms.

10. **Wrap**
   - Show Settings, Dark mode toggle.
   - End.
